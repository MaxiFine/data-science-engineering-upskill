# TYPICAL ANALYTICAL PROJECT PATH

- Background[ Q's like purpose and framing like why, who's my audience]
- Data [what time periods, data source, can the data be trusted]
- Analysis[What have been learnt from the data, insights and interpretations in the data, patterns in the data, key drivers]
- Findings[Clarity & Structure what is the one things to say to audience, does it make sense? ]
- Next Step[Resulting implications, so what? what is the risk in not taking actions]
